Generate Final Synthesis

Consolidates all per-session findings into a single research report. Draws on the theme database for cross-participant patterns and curated quotes, and the per-session analytical narratives for interpretive context from each interview.

Output is a structured Markdown report organised by research question, with verbatim quotes attributed by participant ID. Best run after all transcripts have been synthesised.

Sonnet for most studies. Opus for complex studies with 30+ themes.
