About This Pilot

🎯

Purpose

These tools support the Service Design and Change team's HCD Pilot — helping practitioners move faster from research to insight to action.

🔒

Privacy & Data

All tools run entirely in your browser. No data is sent to external servers unless you explicitly connect an AI API key. Transcripts stay on your device.

📁

Sharing & Handoff

Each tool has an Export JSON button. Upload exports to the team's SharePoint project folder so teammates can pick up where you left off. See the team convention guide for the full workflow.

🚀

Getting Started

Select a tool above to begin. Each tool includes built-in guidance. For questions or feedback, contact the SDC team on SharePoint.
